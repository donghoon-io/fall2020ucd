New 서비스 아이디어

  1. 정보 기입 & 일정 수집 : 현재 거주 나라, 유학하는 나라, 지역, 선호하는 생활 방식 (현재 거주 나라 시간대, 유학하는 나라의 시간대), 수업 시간표 업로드
  1. 현재 생활 습관 진단 하루 평균 커피를 몇 잔 마시는지? 식사가 규칙적이라고 생각하는지? 하루 평균 몇 끼를 먹는지? 일주일에 운동을 평균 몇 회 & 몇 분씩 하는지? 핸드폰 평균 스크린타임은 몇 시간인지? 원하는 최소-최대 수면시간은? 하루 평균 수면시간은?
  1. 수면 패턴 및 생활 패턴 추천 : 생활습관 진단을 바탕으로 사용자의 생활습관 문제점 (카페인 줄이기, 운동 하기, 스크린타임 줄이기...) 2~3개 도출 → 가장 해결하고 싶은 문제점 선택 → 선택한 문제점과 원하는 수면시간 기반으로 생활습관 추천 (수면시간, 식사시간) → 선택한 것을 시간표 (1주일 단위) 사진으로 자동 입력 → 2주 간의 목표(해결해야 할 문제점) 확정
  1. 루틴 확정 : 추천 생활습관 기반으로 2주 간 지켜야 할 루틴 제안 (매일 조금씩 변화하는 루틴 리스트 overview) → Daily Task (루틴 / 개인적 task) 설정
  1. Daily Tracking 자기평가(반원형 시간표) : 추천 생활 습관 기반으로 daily 시간표 제공

    (시간표 뒷 배경 우주에 유저가 일어나있는 시간에 함께 깨어있는 사람들이 얼마나 되는지를 별로 표현하기)

    수면 : 본인이 선택한 생활패턴에 맞게 잘 잤는지(핸드폰, 워치 기록 기반 연동 + 쪽잠의 경우 개별 기록)

    식사 : 몇 끼 먹었는지?

    해결하고 싶은 문제점 : 루틴 제공 (시스템이 진단한 바를 바탕으로 제공하는 것, 매일 난이도 조정)

    개별 Task(체크리스트 개념) : 자신이 입력한 개별 task check (ex. 과제) - 가볍게 언급. (유저에게 커스터마이징할 수 있는 권한 준다)

    → 2주 중 며칠 달성했는지 성취율 보여주기. 3일 연속 하지 않았을 경우 아래와 같이 상대방에게 연대책임

    → 2주 선정 이유 : 문제를 해결하기에는 최소 2주의 시간이 필요 ↔ 너무 길어지면 사용자가 시스템을 사용하는 데에 질릴 수 있다

  1. Mate와 연대책임 : 한 명의 짝을 정해서, 태스크를 수행하지 않았을 경우 다음과 같이 페널티를 주기
    • 수행할 태스크 중, 수행하지 않은 부분이 생겼을 경우 메이트에게 해당 내용을 알림으로 보낸다
    • 알림을 본 메이트는, 알림을 눌러 앱으로 들어가서 대상자의 휴대폰에 "쫌 해"를 보낸다
    • 그러면, 대상자의 휴대폰에는 친구 얼굴이 들어간 해 이미지가 일정 시간 떠있게 된다
    • 이 때, 대상자는 메이트에게 '이러이러한 이유로 못했어', '앞으론 이러이러하게 잘 할게' 등의 대화를 나눈 다음, 메이트가 만족스런 답변이 올 경우에는 쫌 해를 해제해준다
